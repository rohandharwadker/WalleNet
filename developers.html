Hardware Projects

Radiation Detection

Using an Arduino and a Geiger counter, this project is a safety net you'll hopefully never need. While a Geiger counter itself makes electronic 'clicks' to warn the user of radiation, this project includes a 0.96-inch OLED display to accurately and efficiently display the threat of dangerous radiation through CPMs (Counts Per Minute) of alpha and beta particles (particles that occur with nuclear radiation).

Key Points

  • This project premieres the first Arduino-based project, as opposed to the Raspberry Pi. While the Raspberry Pi is a full-fledged computer, the Arduino is simply a micro-controller, making it cheaper but less powerful
  • An OLED display consumes a small amount of power since 'true black' pixels can be turned off to create a more vibrant black and save power
  • A 24000 mAh solar-charged power bank supplies the geiger counter with approximately 30 hours of on-time on a full charge

Motion Camera

Using a Raspberry Pi 4B+, a camera, and a PIR motion sensor, this motion camera is lightweight, small and discrete. The concept is quite simple: it will record a video when motion is detected and alert the user.

Key Points

  • Since the Pi 4B+ has a built-in WIFI module, the camera can alert you from the other side of the Earth!
  • The video recorded is emailed to you as soon as it is captured, and can also be uploaded to dropbox. This means once you place the camera, you'll never have to worry about 'retrieving the footage' and then having to re-position the camera
  • The Pi is powered by a 5v USB-C cable, meaning it will be easy to install connecting to a usb port

International Space Station Tracker

Verison 1.3

A simple python script with a GUI and Terminal output showing the relative position of the International Space Station on a map using data from api.open-notify.org. The terminal also displays the names of all people currently aboard the ISS.

AutoGit

Verison 2.0

AutoGit is a Git Automator that allows you to push changes to your git repository with an easy-to-use GUI interface. This can make pushing to your repo much less of a daunting task and is good for beginners who don't want to interact with the GIT Terminal interface.
